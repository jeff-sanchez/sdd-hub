O que é
SDD?
Spec-Driven Development é uma metodologia que elimina o caos do vibe coding. Cada fase produz um contrato que a próxima deve respeitar — sem pular etapas, sem alucinações, sem resultados imprevisíveis.
Por que isso importa
O vibe coding cria um loop caótico: prompt → código → não funcionou → edita prompt → código diferente → ainda não funciona. Isso é aceitável para protótipos descartáveis. Para sistemas reais com dados reais e usuários dependentes, é inaceitável.
